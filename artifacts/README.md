# Artifacts

This directory contains compact artifacts that support the paper and README.

- `tables/`: processed CSVs and printed paper values.
- `analysis/`: checked-in EN-ZH embedding-space diagnostics.
- `examples/`: small example result bundles.

The released word-mix query bundles are hosted separately on Hugging Face rather than in this directory:

- [hcm777/query-embedding-mix-word-mix](https://huggingface.co/datasets/hcm777/query-embedding-mix-word-mix)

Large raw outputs are intentionally not stored here. See `docs/ARTIFACTS.md` for the full artifact policy.
